# Atalhos de Teclado — DBeaver

Referência rápida de atalhos para uso com teclado no DBeaver. Testado com NVDA.

> Este arquivo é um resumo navegável. A versão completa e oficial está disponível em PDF, na seção de releases deste repositório. Licença: CC BY-NC-ND 3.0 BR.

---

## Navegação Geral

| Atalho | Ação |
|---|---|
| `Alt` | Acessar a barra de menus |
| `Alt + F4` | Fechar o DBeaver |
| `Ctrl + W` | Fechar a aba/edição atual |
| `Ctrl + N` | Nova conexão / novo arquivo |
| `Ctrl + Q` | Abrir a caixa de busca rápida |
| `Ctrl + Shift + L` | Lista de atalhos do DBeaver |

## Editor SQL

| Atalho | Ação |
|---|---|
| `Ctrl + Space` | Autocompletar código SQL |
| `Ctrl + Enter` | Executar a linha ou seleção |
| `F3` | Ir para próxima ocorrência |
| `Shift + F3` | Ir para ocorrência anterior |
| `Ctrl + /` | Comentar/descomentar linha |
| `Ctrl + Shift + /` | Comentar/descomentar bloco |
| `Ctrl + D` | Duplicar linha |
| `Ctrl + Z` | Desfazer |
| `Ctrl + Shift + Z` | Refazer |

## Navegação em Resultados

| Atalho | Ação |
|---|---|
| `Ctrl + Enter` | Executar consulta |
| `F5` | Atualizar resultado |
| `Ctrl + F` | Buscar no resultado |
| `Setas` | Navegar entre células |
| `Home` / `End` | Ir para início/fim da linha |
| `Ctrl + Home` / `Ctrl + End` | Ir para início/fim do resultado |

## Conexões e Projetos

| Atalho | Ação |
|---|---|
| `Ctrl + Shift + B` | Abrir Painel de Banco de Dados |
| `F2` | Renomear item selecionado |
| `Delete` | Excluir item selecionado |
| `Shift + F2` | Propriedades da conexão |

## Acessibilidade

- O DBeaver é baseado em Eclipse RCP, que possui suporte nativo para leitores de tela.
- Use `Alt` para navegar pelos menus e `Tab` para percorrer campos de formulário.
- A barra de status do DBeaver é anunciada pelo NVDA quando há mudanças de estado.
